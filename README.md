## üîß How to

1. clone repo
2. install the dependencies (backend / frontweb)
```bash
npm install
```
3. run the project

backend
```bash
npm run dev
```
frontweb
```bash
npm start
```

4. login
- email: `user@email.com`
- password: `123456`
